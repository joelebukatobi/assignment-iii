Input Date

Date in HTML is a type or attribute that's specific to the <input type="date"> element. It defines a date with no timezone and it's output varies from one browser or platform to another

Input Datetime

Datetime in HTML is a type or attribute that's specific to the <input type="datetime"> element. It defines a control for entering a date and time hour, minute, second and fraction of a second) as well as timezone, but its been deprecated

The feature is obsolete even though it might still work in some old browsers

Input Datetime-Local

Date in HTML is a type or attribute that's specific to the <input type="datetime-local"> element. It defines input controls that let the user easily enter both a date and a time, including the year, month, and day as well as the time in hours and minutes.

Input Month

Date in HTML is a type or attribute that's specific to the input element. It defines controls for <input type="month"> fields that let the user enter a month and year allowing a month and year to be easily entered. The value is a string whose value is in the format "YYYY-MM", where YYYY is the four-digit year and MM is the month number.

Input Week

Date in HTML is a type or attribute that's specific to the <input type="week"> element. It defines control for input fields of the type week that allows easy entry of a year plus the week number during the year

Textarea

The <textarea> is one of the form element and is more often than not nested inside of a <form> element. it is a multiline plain-tex editing control, that is mainly used to allow users enter a sizeable amount of text

Drop Down List (Multiple Select)

The <select> is another form element and is used as an input option providing a multitude of options;

HTML Validation

HTML Validation refers to the very first validation done before client side validation. This is achieved using an attribute required which is mainly used with the <input> tag as well as other form elements such as the <textarea> element and the <select> element

File Upload

In HTML File uploads are done using the <input> element with the type of file it also has other attributes such as accept that determine the file type that can be uploaded

HTML Entities

HTML entities refer to a piece of string or text that generally begin with an (&) and ends with a semicolon (;) These entities are used to display reserved characters. for instance the coypright © is &copy; and the $ is &dollar;

Quotation and Citation

In HTML quotations generally come in two formats, the <blockquote> which is used to indicate an enclosed text in an extended quotation which more often than not inludes a source which may be given using the <cite> attribute;

The second is the <q> and it is intended for short quotations that don't require paragraph breaks and is usually denoted with quotes

Table

A table in HTML is usually denoted by the <table> element which is used to represent data in a tabluar form using rows and columns. it has two major row elements which are <thead> and <tbody> they both contain the <tr> element which in turn houses the <th> for the <thead> and <td> for the <tbody>
